## 🚀 快速開始

1. **推送到 GitHub**（2 分鐘）
   - 在 GitHub 上建立新倉庫 `ai-pastor`
   - 執行：`git remote add origin https://github.com/YOUR_USERNAME/ai-pastor.git && git push -u origin main`

2. **部署後端到 Railway**（5 分鐘）
   - 前往 https://railway.app/
   - 用 GitHub 登入
   - 部署您的倉庫
   - 設定環境變數 `GOOGLE_API_KEY`
   - 獲得後端 URL

3. **更新前端配置**（1 分鐘）
   - 將後端 URL 更新到 `index.html`
   - 重新建置並推送

4. **部署前端到 Netlify**（3 分鐘）
   - 前往 https://www.netlify.com/
   - 用 GitHub 登入
   - 導入專案並部署
   - 獲得前端 URL（這就是公開連結！）
